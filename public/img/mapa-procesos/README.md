# Diagramas del mapa de procesos

Guarda aqui los diagramas y manten el nombre listado en `diagramasPorProceso` dentro de `MapaProcesos2025.vue`. Esto permite conservar los nombres descriptivos actuales, incluidos espacios y acentos.

Ejemplo actual:

`/img/mapa-procesos/E1. DIRECCIONAMIENTO ESTRATEGICO.png`

Para agregar un proceso sin diagrama, primero agrega su archivo a esta carpeta y luego incorpora su ruta en ese mapa de nombres.
